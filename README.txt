#. 앱 이름
PhoneBook_App

#. 앱 개요
PhoneBook_App은 연락처 생성하고, 연락처에 등록된 대상자들 간의 호출을 지원하는 애플리케이션입니다. 이 애플리케이션의 주요 기능은 연락처 조회 및 추가, 상대에 대한 알림 생성입니다.

#. 앱 기능
- 사용자 연락처 조회 : 앱에 등록된 다른 사용자의 정보에 대한 조회가 가능합니다.
- 사용자 연락처 추가 : 이름, 전화번호, 플레이 중인 게임 제목을 입력하여 새로운 연락처를 추가합니다.
- 사용자 연락처 업데이트 : 앱 시작 시, 휴대폰에 등록된 기존 연락처에 대한 정보와 기존에 앱에 등록된 정보를 소스로 앱의 목록을 구성합니다.
- 사용자 연락처 수정 : 앱에 등록된 자신의 정보를 수정합니다.

#. 설치 및 실행 방법
1. 해당 깃 저장소의 프로젝트를 다운로드합니다.
2. 다운로드가 완료된 프로젝트를 안드로이드 스튜디오로 빌드합니다.
3. 안드로이드 스튜디오를 활용하여 자신의 스마트폰에 프로젝트를 빌드, 실행합니다.
